            This extension does nothing on its own; you also need to install and run the
            text-aid-too server.  See the project's home page (linked below) for details.
          
    • 
          
  • 
            There's no need to save your changes; just use Escape to blur the inputs above.
            Option values are synchronised between Chrome instances.
          
    • 

          
  • 
            Set Port to the port number used by the server.  The default is 9293.
          
    • 

          
  • 
            Set Secret to any secret you provided to the server.  If you provide a secret to the
            server (via the $TEXT_AID_TOO_SECRET environment variable), then it will ignore
            requests that don't contain that secret.
          
    • 

          
  • 
            The default keyboard shortcut is <Ctrl-;>, but you can set your own shortcut
            above.
